## Installation

inmobi-ios-sdk is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'inmobi-ios-sdk', :git => 'https://github.com/freezy7/inmobi-ios-sdk-pod.git'
```
